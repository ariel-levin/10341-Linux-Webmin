Linux Webmin
December 2014

Final Project of "Operating Systems UNIX & Linux" course (code 10341), during our Software Engineering B.Sc. at Afeka Academic College of Engineering, Tel Aviv, Israel.

The Project included developing a Webmin interface for system administration of a remote Linux server, using PHP and Shell scripts on the server side, and HTML, JavaScript and jQuery on the Client side.
